# DVLD v5.3
Driving and Vehicle License Management system (C#.NET WinForms , ADO.NET , MSQL Server):
- App can easily be altered to support other management systems & tailored to specific client needs,
- Built DAL APIs using MSQL & Reduced overhead on db. calls by using ADO.NET for Database access,
- Handled user authentication on client machine using windows Registry,
- Built a WinForms Frontend Interface and deployed the app over local network on shared folder (FTP Protocol)
and used my PC as Central Database server,
- Working on adding database insertion & deletion triggers for daily data reporting.
